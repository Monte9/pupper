# Pupper
### Using artificial emotional intelligence to help you find your new furry best friend!

Built with love at TreeHacks 2017

## How to get started:

1. `git clone https://github.com/morendawg/pupper`
2. `cd pupper`
3. `yarn`
4. `npm run build`
5. `node server`
